### 哥特式衬线标题字体

- **授权类型**: SIL Open Font License (OFL) 1.1

- **来源URL**: https://fonts.google.com/specimen/MedievalSharp

- **授权类型**: SIL Open Font License (OFL) 1.1
- **是否允许免费商用**: ✅ 是
- **是否需要署名**: ❌ 不需要
- **风格匹配度**: ⭐⭐⭐⭐⭐ (5/5)
- **尺寸/格式**: TTF / OTF
- **备注**: Blackletter-adjacent风格但保持可读性。带有中世纪手稿的韵味，非常适合暗黑幻想主题。Google Fonts中少有的真正"中世纪"风格字体。适合副标题和章节标题。

## 注意事项与后续建议

### 1. 字体使用建议

- **Google Fonts字体**是最安全的选择：全部开源授权，商用无忧，无需署名。
- **中文支持**: 如果游戏需要中文界面，Noto Sans SC/TC是最佳选择。如果仅英文，选择更多。
- **像素字体显示技巧**: Press Start 2P、Dogica等像素字体在**特定尺寸**（如8pt, 16pt, 24pt）显示效果最佳，缩放至非整数倍可能导致模糊。在Godot中建议开启"Nearest"过滤模式。
- **字体配对建议**:
  - 标题: Cinzel (主标题) + MedievalSharp (副标题)
  - 正文: Noto Sans Regular
  - 伤害数字: Bangers + 红色/黄色渐变
  - 数值面板: Space Mono
  - 按钮: Noto Sans Bold / Roboto Medium

### 2. 特效使用建议

- **调色适配**: 很多特效资源原始色调可能不完全匹配暗黑哥特主题。建议在Aseprite或GIMP中：
  - 降低饱和度 → 增加暗黑感
  - 偏蓝/紫色 → 增加神秘感
  - 增加对比度 → 增加冲击力
  - 反相 → 某些特效可变成"暗影"版本
- **尺寸处理**: 如需精确尺寸（如32x32），可从较大资源（64x64）缩放，或使用Aseprite重新导出。
- **帧率调整**: Godot中动画速度可调，特效素材的原始帧率无需严格匹配。
- **CC0 vs CC-BY**: 优先使用CC0资源（如Kenney、Undead Pirate Roguelike、Animated Particle Effects），完全无需担心署名。CC-BY资源需在游戏 credits 中正确标注作者。
