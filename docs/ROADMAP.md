# 后续开发路线图

## 🔴 Phase 12: 美术资源替换

### 现状
当前所有精灵图使用程序化生成的彩色方块，视觉效果较差。

### 目标
替换为像素美术资源，提升游戏视觉质量。

### 具体任务

#### 12.1 角色精灵 (高优先级)

| 资源 | 尺寸 | 动画帧数 | 描述 |
|------|------|----------|------|
| 玩家待机 | 32x32 | 4帧 | 呼吸/浮动效果 |
| 玩家移动 | 32x32 | 6帧 | 跑步动画 |
| 玩家攻击 | 32x32 | 4帧 | 射击动作 |
| 玩家受伤 | 32x32 | 2帧 | 受伤反馈 |
| 玩家死亡 | 32x32 | 6帧 | 死亡动画 |

#### 12.2 敌人精灵 (高优先级)

**基础敌人**:
- 🦇 蝙蝠 - 24x24，4帧飞行
- 🧟 僵尸 - 36x36，6帧行走
- 🐺 狼人 - 40x40，待机+冲锋
- 🧛 吸血鬼 - 32x32，待机+移动
- 💀 骷髅射手 - 32x32，待机+射击

**Boss**:
- 👹 吸血鬼领主 - 64x64，6组动画（待机、弹幕、召唤、冲刺、狂暴、死亡）

#### 12.3 UI 元素 (高优先级)

- 按钮（正常/悬停/按下状态）
- 血条、经验条
- 升级卡片
- 武器/被动道具图标

#### 12.4 背景与环境 (中优先级)

- 房间背景图
- Tilemap 瓦片
- 环境装饰（火炬、窗户、棺材等）

#### 12.5 特效 (中优先级)

- 受击粒子效果
- 死亡爆炸效果
- 升级光环
- 拾取特效

> 详细规格见: [docs/art-assets-requirements.md](docs/art-assets-requirements.md)
> 快速入门见: [docs/art-development-guide.md](docs/art-development-guide.md)

## 🟡 Phase 14: 玩法扩展

### 14.1 新增敌人

- **自爆者**: 接近玩家后爆炸
- **召唤师**: 召唤小怪
- **精英怪**: 带词缀（速度/护甲/分裂）
- **石像鬼**: 石化/苏醒机制
- **血族亲王**: 高级敌人

### 14.2 新增武器

- 🔥 火焰喷射器 - 持续伤害
- 🪃 苦无 - 快速投掷
- 💥 血爆 - 范围爆炸

### 14.3 新增角色

| 角色 | 定位 | 特性 |
|------|------|------|
| 刺客 | 高速低血 | 高攻速、高闪避 |
| 法师 | 远程AOE | 魔法伤害、范围攻击 |
| 骑士 | 近战坦克 | 高血量、高护甲 |

### 14.4 被动道具扩展

- 更多被动组件
- 被动道具联动效果

### 14.5 成就系统

- 击杀里程碑
- 通关挑战
- 隐藏解锁

## 🟢 Phase 15: 关卡深度

### 15.1 多层地下城

- 每 10 个房间进入下一层
- 敌人强度递增
- 难度曲线设计

### 15.2 地图事件

- 随机事件（陷阱/NPC/隐藏房间）

### 15.3 多 Boss

- 每层一个 Boss
- 最终 Boss: 吸血鬼女王
- Boss 技能库扩展

### 15.4 每日挑战

- 固定种子
- 特殊规则

## 🎯 建议开发顺序

1. **立即**: 替换关键 UI 元素（提升可玩性）
2. **短期**: 替换角色和敌人精灵（提升视觉体验）
3. **中期**: 添加音效/音乐（提升沉浸感）
4. **长期**: 玩法扩展和关卡深度（增加可玩内容）
