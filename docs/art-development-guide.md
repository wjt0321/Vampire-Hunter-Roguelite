# 美术开发指南

> 本指南帮助美术人员快速了解游戏视觉风格和开发优先级

---

## 🎮 游戏概述

- **类型**: 2D 俯视角 Roguelite 动作游戏
- **风格**: 暗黑哥特 + 像素艺术
- **分辨率**: 1280x720（游戏内渲染，可缩放）
- **视角**: 固定俯视角

---

## 🖼️ 视觉风格

### 色彩方案

| 用途 | 主色 | 色值 |
|------|------|------|
| 背景 | 深紫黑 | `#1A0A1A` |
| 玩家 | 猎人棕 + 暗红披风 | `#E8D5B7`, `#8B0000` |
| 敌人 | 暗红 + 紫色 | `#4A0000`, `#2D0047` |
| UI | 金色 + 深红 | `#FFD700`, `#8B0000` |
| 特效 | 橙红 + 紫罗兰 | `#FF4500`, `#9400D3` |

### 像素风格规范

- **精灵尺寸**: 32x32（角色）, 16x16（小物件）
- **帧率**: 8-12 FPS
- **描边**: 2px 黑色轮廓
- **色深**: 32-64 色调色板

---

## 📋 开发优先级

### 第一阶段：核心角色（高优先级）

需要为以下内容制作像素精灵：

#### 玩家
- 猎人阿尔忒弥斯 - 32x32，6 组动画
  - 待机、跑动、射击、受伤、死亡、护盾环绕

#### 敌人
| 敌人 | 尺寸 | 动画 |
|------|------|------|
| 🦇 蝙蝠 | 24x24 | 飞行 |
| 🧟 僵尸 | 36x36 | 行走 |
| 🐺 狼人 | 40x40 | 待机、冲锋 |
| 🧛 吸血鬼 | 32x32 | 待机、移动 |
| 💀 骷髅射手 | 32x32 | 待机、射击 |
| 👹 吸血鬼领主 (Boss) | 64x64 | 待机、弹幕、召唤、冲刺、狂暴、死亡 |

### 第二阶段：UI 元素（高优先级）

- 按钮（正常/悬停/按下）
- 血条、经验条
- 升级卡片
- 武器/被动道具图标
- 商店界面

### 第三阶段：环境和特效（中优先级）

- 房间背景图
- Tilemap 瓦片
- 粒子特效序列帧

### 第四阶段：音频（低优先级）

- BGM（主菜单、战斗、Boss战）
- SFX（射击、命中、死亡、升级）

## 📁 文件命名规范

```
[类型]_[名称]_[状态].[格式]

示例：
player_idle.png
zombie_walk.png
boss_attack1.png
icon_pistol.png
hp_bar_fill.png
```
